Features

Assign students

Retrieve and lists all students you are teaching.

Assign the students you want to tasks as you see fit based on your needs

Generate groups

Generate groups based on the students you have assigned and the number of students you wish to assign on the task.

While students are randomly distributed inside the created groups you can enforce a balanced distribution based on specific attributes for better results.

Customize groups

Generated groups can have customed names if you so wish but default names will be used if this isn't needed.

After generating groups it is possible to manually reassign students between groups if the need arises
